50 // Logger class
51 /*
52 ------
53 Notes:
54 ------
55 - This class is meant to be a high speed signal logger with low computational
56 impact on the application thread. The logger writes to file on a separate
57 thread.
58 - Variables to be logged are registered before the logger starts. The variables
59 have to persist across the lifetime of the logger, otherwise a segfault will
60 occur.
61 - Eigen vector and matrices, double, int and bool are supported
62 - Time since start is automatically logged as the first column of the output
63 file.
64 - Logging frequency is specified in Hz and is 100 Hz by default.
65 - Output is a csv file with following format, and the variables will always be
66 logged with the eigen values first, followed by double, int and bool values:
67 
68 time, Vector1_0, Vector1_1, Vector2_0, Vector2_1, double1, int1, bool1
69 0, 5.0, 5.0, 5.0, 5.0, 2.0, 23, 0
70 0.01, 5.0, 5.0, 5.0, 5.0, 2.0, 23, 0
71 0.02, 5.0, 5.0, 5.0, 5.0, 2.0, 23, 0
72 ...
73 */

151  }
