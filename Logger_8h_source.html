53 /*
54 ------
55 Notes:
56 ------
57 - This class is meant to be a high speed signal logger with low computational
58 impact on the application thread. The logger writes to file on a separate
59 thread.
60 - Variables to be logged are registered before the logger starts. The variables
61 have to persist across the lifetime of the logger, otherwise a segfault will
62 occur.
63 - Eigen vector and matrices, double, int and bool are supported
64 - Time since start is automatically logged as the first column of the output
65 file.
66 - Logging frequency is specified in Hz and is 100 Hz by default.
67 - Output is a csv file with following format, and the variables will always be
68 logged with the eigen values first, followed by double, int and bool values:
69 
70 time, Vector1_0, Vector1_1, Vector2_0, Vector2_1, double1, int1, bool1
71 0, 5.0, 5.0, 5.0, 5.0, 2.0, 23, 0
72 0.01, 5.0, 5.0, 5.0, 5.0, 2.0, 23, 0
73 0.02, 5.0, 5.0, 5.0, 5.0, 2.0, 23, 0
74 ...
75 */
